# ByReiXwift

ByReiXwift is a Shariah-compliant online payment system designed for the global Muslim community. It prioritizes ethical finance by eliminating interest (Riba) and ensuring absolute transparency in transaction fees.

## Key Principles

The platform follows strict Islamic finance foundations—eliminating ambiguity (Gharar) and interest (Riba).

- **Zero Interest**: No interest charges on transactions or balances.
- **Transparency**: Fixed fee structures with no hidden costs.
- **Ethical Integrity**: Verification protocols for all partnerships and transactions.

## Development Setup

### Prerequisites
- Node.js (Latest LTS)
- pnpm

### Quick Start
```bash
git clone <repository-url>
cd byreixwift
pnpm install
pnpm dev
```
The application will be available at `http://localhost:3000`.
